# getopt
![](http://savepic.ru/8544715.png)

В связи с тем, что задание может трактоваться неоднозначно, зададимся несколькими условиями:
-Параметр --help перекрывает действие остальных параметров
-Аргумент параметра всегда отделяется от других аргументов пробелами, либо знаком "="
Корректно: 
--a--b 2 --c
--a --b=2 --c
Не корректно:
--a--b 2--c
-Будем считать, что при вводе не происходит ошибок. Не будем обрабатывать некорректный ввод.
-При выводе подсказки будем обозначать возможность выбора одного из элементов набором знаков {a| b| 3}
-Аргумент параметра заносится в глобальную переменную arg.
